About


The Hardy-Weinberg Equilibrium (HWE) assumption is essential to many population genetics models, which assumes
that allele pairs from a given population are random. An HWE test needs to test whether the pairing are random or not.

The algorithm was developed by Or Shkuri and Prof. Yoram Louzoun at Bar-Ilan University.
This website was implemented by Yuli Tshuva.
A standalone version code is available at the GitHub repository.
Visit Prof. Louzoun's lab website to further explore the lab's research domains.
